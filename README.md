# 🎵 Spotify UI Clone with CI/CD and Docker

A responsive Spotify web player UI clone built using HTML and CSS, enhanced with Docker containerization and CI/CD pipeline integration.

## 🚀 Features
- Responsive UI replicating Spotify layout and design
- Modular components: sidebar, playlists, cards, and music player
- Containerized using Docker for consistent deployment
- CI pipeline implemented using GitHub Actions
- Deployed on Netlify for public access

## 🛠️ Tech Stack
- HTML
- CSS
- Docker
- GitHub Actions (CI/CD)
- Netlify (Deployment)

## ⚙️ DevOps Workflow
- Version control using Git and GitHub
- Docker used to containerize the application
- GitHub Actions used to automate build process on code push
- Continuous deployment via Netlify

## ▶️ Run Locally with Docker
```bash
docker build -t spotify-clone .
docker run -d -p 8080:80 spotify-clone
```

Then open:
http://localhost:8080
